# instagram
使用flask仿instagram



## 项目介绍
   使用flask仿instagram,阉割了很多功能
   
   
   
## 所用技术

 * 数据库
    * sqlite
    * 七牛云
 * 数据库访问
   * flask-sqlalchemy
   * 七牛sdk
 * web框架
   * flask
 * 测试
   * unittest 
   
## 安装依赖

~~~
        pip install -r requirements.txt
~~~
   
## 重建数据库

~~~
        python manage.py
~~~

## 运行web App 

~~~
        python serverrun.py
~~~
